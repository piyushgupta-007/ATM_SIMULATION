# ATM_SIMULATION
A simple ATM simulation system in Python with account creation, login, deposit, withdrawal, balance enquiry, and mini statement.

Features
- Create New Account
- Secure Login with 4-digit PIN
- Deposit Money
- Withdraw Money
- Balance Enquiry
- Mini Statement (Last 10 Transactions)
- Change PIN
- JSON-based Data Storage
- Fully Menu-Driven CLI Interface

Technologies Used
- Python 3
- JSON Storage
- Datetime Module
- OS Module
  
Testing Instructions
- Create a new user account
- Login using Account Number & PIN
- Perform deposit, withdrawal
- Check balance
- View mini statement
- Change PIN and verify
- Check JSON file updates
